**Project Overview**
The Digital Student Portfolio System is a JavaFX-based desktop application designed to help students create, manage, and submit their academic portfolios digitally.
It also provides functionality for faculty members to review submissions and for administrators to manage the system efficiently.
This project follows a clean MVC architecture with a dedicated service layer, making it scalable, maintainable, and suitable for real-world academic environments.
**Features**
✅Student
✔️Create and update personal portfolios
✔️Submit portfolios for faculty review
✔️View submission status and feedback
✅Faculty
✔️Review submitted student portfolios
✔️Approve or reject submissions
✔️Provide feedback
✅Administrator
✔️Manage users and portfolios
✔️Access admin dashboard
✔️Monitor system activities
✅Project Architecture
The project follows MVC + Service Layer architecture:
com.amazi
